# BossNeedMore
一个无须任何额外环境(除非你没有浏览器)的小项目，其源于我的一位爱画画朋友需要找一下学习素材。你可以使用其获取一些精美的图片(感谢各位大佬提供的图片接口)。

BossNeedMore 使用说明
恭喜你成功解压了压缩包，打败了 99% 的用户！

简介
本程序的初衷是让我的老大能快捷地找到绘画所需的参考图（所以叫 BossNeedMore）。

本程序是一个纯前端的静态网页程序，无须任何依赖或软件（如果你没有浏览器的话，当我没说）。

配置说明
你可以在 config.json 中修改程序的配置以增加或减少图片源（或联系陈某帮忙操作）。

配置参数说明如下：

参数	说明
text	图片源的文本描述。
value	图片源地址。为了保证每次获取的图片尽可能不同，建议在本参数的最后额外添加 ?{{timestamp}}，其中的 {{timestamp}} 会被自动替换为当前时间戳。
clickAction	图片的点击行为。
- "none"：表示不做任何反应。
- "fullscreen"：表示全屏显示。
showDownload	是否显示下载按钮。
- "true"：显示。
- "false"：不显示。
一般因为各种原因，本程序不可下载，或下载不稳定的源，建议设为 "false"。

快速开始
1. 新建快捷方式
初次运行时，请点击 setup.bat，这会帮你自动新建一个快捷方式。

2. 启动程序
以下三种方法都可以打开程序：

点击桌面快捷方式。
直接双击打开 index.html。
运行 launcher.bat。

操作指南
选择图片源：打开程序后，通过下拉菜单选择你需要的图片源。
获取图片：点击 “获取新图片” 按钮，即可获取一张图片，图片会显示在下方的图片框中。
清空图片：点击 “清空图片” 按钮，即可清空图片框。
下载图片：点击 “下载图片” 按钮，即可下载当前显示的图片（如果配置中 showDownload 设为 "false"，则此按钮不会显示）。
全屏查看：点击显示的图片，即可全屏显示（如果配置中 clickAction 设为 "fullscreen" 的话）。
